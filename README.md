## 项目特点

- ✅ 支持Windows/Linux/Mac全平台运行
- 🚀 GitHub Actions自动构建和发布
- 📦 自动生成平台特定的可执行包
- ⚡ 一键运行的启动脚本

## 快速开始

### 下载和使用

1. 从 [Releases](https://github.com/levixie2015/ghactions/releases) 页面下载最新版本
2. 选择对应系统的包：
   - Windows: `release-windows.zip`
   - Linux: `release-linux.tar.gz`
   - macOS: `release-mac.tar.gz`
3. 解压下载的文件
4. 运行启动脚本：
   - Windows: 双击 `run.bat`
   - Linux/Mac: 执行 `./run.sh`
